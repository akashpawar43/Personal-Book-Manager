<h1 align="center" id="title">Personal Book Manager</h1>

<p id="description">Personal Book Manager App built using the MERN Stack, with Next.js, Redux Toolkit, and Thunk API. It allows users to create, update, and delete books to manage their personal book collection.</p>

  
  
<h2>🧐 Features</h2>

Here're some of the project's best features:

*   Add Book
*   get Books
*   Update Status
*   Delete Book


<h2>🛠️ Installation Steps:</h2>

<p>1. cd Server</p>

<p>2. Install dependencies for Server</p>

```
npm install
```

<p>3. Start Server</p>

```
npm run dev
```

<p>4. Open Separate terminal for Client</p>

<p>5. cd Client</p>

<p>6. Install dependencies for Client</p>

```
npm install
```

<p>7. Run Client</p>

```
npm run dev
```
